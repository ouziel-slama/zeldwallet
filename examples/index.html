ZeldWallet examples

Run npm run dev from the repo root, then open http://localhost:5173/. Pick any demo below.

Web component

Showcases the native <zeld-wallet-ui> element with language toggle, dark mode styles, and message signing.

sats-connect integration

Demonstrates wallet discovery via WBIP004 and sats-connect, includes provider registration and request/response logging.

React demo

Renders the <ZeldWalletCard> wrapper, toggles language + variant, and signs a message once the wallet is unlocked.

Tip: the demos load dist/zeldwallet.es.js. Run npm run build after changing the library to refresh the bundle.
